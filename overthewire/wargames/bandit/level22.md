```sh
bandit22@bandit:/etc/cron.d$ cat /usr/bin/cronjob_bandit23.sh
#!/bin/bash

myname=$(whoami)
mytarget=$(echo I am user $myname | md5sum | cut -d ' ' -f 1)

echo "Copying passwordfile /etc/bandit_pass/$myname to /tmp/$mytarget"

cat /etc/bandit_pass/$myname > /tmp/$mytarget
```

This script creates a tmp file with a name that is generated by the command `echo I am user $myname | md5sum | cut -d ' ' -f 1`. We know that `$myuser` = `bandit23`, so we can just run the same command with that name substituted for ourselves, and receive the secret file.
